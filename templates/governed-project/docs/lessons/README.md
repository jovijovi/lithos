# Lessons directory

Use this directory for reusable lessons learned from concrete project work.

Naming convention:

```text
docs/lessons/YYYY-MM-DD-topic.md
```

Each lesson should include frontmatter with `title`, `status`, `created_at`, and, when drift should be tracked, `last_validated_at` plus `applies_to` paths.
